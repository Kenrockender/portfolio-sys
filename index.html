Kenapa PORTFOLIO.SYS?

Bukan sekadar spreadsheet.

Dirancang khusus untuk investor Indonesia yang punya aset di banyak tempat sekaligus.

Harga Live Otomatis
Sync dari CoinGecko, Yahoo Finance & ExchangeRate-API setiap 5 menit. Tidak perlu update manual lagi.
Cloud Sync & Privat
Data tersimpan di Firebase Firestore. Hanya kamu yang bisa akses — tidak ada yang bisa melihat portofolimu.
Import PDF & CSV
Paste atau upload langsung dari Bibit, Stockbit, Pluang, Indodax, Tokocrypto, IBKR, dan banyak lagi. Tidak perlu ketik manual.
Analytics & Benchmark
Sharpe ratio, max drawdown, volatilitas, dan perbandingan performa vs IHSG & S&P 500. Data science untuk investor ritel.
Rebalancing Calculator
Set target alokasi per kelas aset, langsung lihat berapa yang harus dibeli atau dijual untuk mencapai target.
PWA — Bisa Offline
Install di HP seperti app native. Berjalan lancar tanpa koneksi. Data terakhir tetap tersedia walau internet mati.

FAQ

Pertanyaan yang sering ditanya.

Apakah data portofolio saya aman? +
Ya. Data tersimpan di akun Google-mu sendiri melalui Firebase Firestore. Kami tidak bisa melihat, mengakses, atau menjual data portofoliomu. Setiap user punya path database sendiri yang terlindungi oleh Firestore Security Rules.
Bisa dipakai di HP? +
Ya. PORTFOLIO.SYS adalah Progressive Web App (PWA). Kamu bisa install di iPhone atau Android — tampilannya seperti app native, bisa dipakai offline, dan ada notifikasi jika didukung.
Apakah harga akan naik? +
Ya. Lifetime Deal Rp 199.000 hanya untuk 100 pembeli pertama sebagai harga early adopter. Setelah slot habis, harga kembali ke Rp 499.000 atau akan beralih ke model subscription bulanan.
Bagaimana kalau saya tidak puas? +
Ada garansi uang kembali 3 hari sejak pembelian, tanpa pertanyaan. Hubungi kami via email atau DM dan kami proses refund dalam 24 jam.
Apakah ini saran investasi? +
Tidak. PORTFOLIO.SYS adalah alat pencatatan dan visualisasi portofolio. Semua data yang ditampilkan adalah berdasarkan input dari pengguna. Tidak ada dari konten di platform ini yang merupakan saran investasi.
Broker & exchange apa saja yang didukung untuk import? +
Saat ini mendukung import dari: Bibit, Stockbit, IndoPremier, Pluang (CSV & PDF email), Indodax (Tax Report PDF), Tokocrypto (Laporan Pajak PDF), Floq (Laporan Pajak PDF), Binance, IBKR (Interactive Brokers), dan format CSV generik. Daftar terus bertambah.
